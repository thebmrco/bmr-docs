# Release 11 (BETA) 2026 May

This release introduces **BMR Studio** — the unified 3D workspace for all room design work, available on Web, iPhone, and iPad — alongside redesigned **Organization** and **Location** overview pages, **Notes** across all elements on both Web and App, and significant refinements to the **Acoustic Flow**.

**BMR Studio** is now available — the single 3D environment used for all room design work in BMR. It is accessible on the **Web**, **iPhone**, and **iPad**.

**Modes**

BMR Studio uses dedicated modes so you can focus on one thing at a time:

- **View** — explore the room and review designs
- **Structurals** — adjust structural components such as walls, windows, doors, and other building elements
- **Equipment** — add and remove equipment and furniture in the design
- **Cables** — plan and document cable routing between devices

More modes will be added over time, and existing modes will continue to be refined.

**What you can do**

- Place and configure equipment from the catalog
- Create and manage multiple design versions for the same room
- Move a design through approval and mark it as the **Active Design**
- Generate an equipment list automatically from placed items

Open BMR Studio from the **Design** section of any room — on Web, iPhone, or iPad.

Organizations can now define **Default Room Configurations** — recommended equipment setups based on room size and usage type.

For example, you can specify that a room between 20–40 square meters should have a Room Bar, dual 65" screens, a table, and 6 chairs. When a user opens **BMR Studio** to design a room that falls within that size range, they are presented with the recommended configuration and can select and place those components directly into the design.

**How it works**

- Define configurations by **room size range**, **usage type**, and **suggested equipment** with quantities
- Equipment is selected from the full BMR catalog
- When designing a room in Studio, matching configurations appear as **recommendations** in the sidebar
- The user can accept the recommendation and place the suggested items, or choose their own

This makes it easy to maintain consistent standards across an organisation — from 10 rooms to 10,000.

Default Room Configurations are managed under the **Default Room Configurations** tab on the Organization page.

## App (iPhone / iPad)

<details className="bmr-details">

    <Mic size={18} /> Acoustic Measurement Flow Refinements
  </span>

Following the launch of the new Acoustic Flow in Beta 08, this release introduces several reliability and usability improvements.

**What's new**

- The app now **alerts you if no scan exists** before starting a measurement, preventing invalid MOS = 0 results
- You can **retest during the volume adjustment step** without restarting the flow
- A **Close (X) button** has been added to all measurement steps for easier cancellation
- **SNR calibration updated** for more accurate speaker setup
- The acoustic list view now shows the **MOS Score** and **output device** for each measurement

These changes make the Acoustic Flow more forgiving, faster to recover from mistakes, and more accurate end-to-end.

</details>

You can now **place equipment directly in the physical room using AR** and save the result as a design. The AR Design combines with the room scan so you can continue refining in BMR Studio afterwards.

**How it works**

1. **Find coordinates** — move the iPhone around the room so the app can locate the scan's coordinate space
2. **Add equipment** — tap the equipment icon to browse the catalog and select items to place
3. **Choose content** — for screens and displays, select what content to show (still images or video)
4. **Place in the room** — position the item at the right location in the physical space
5. **Save** — upload the AR-placed items as a design
6. **Continue in Studio** — the design appears in the Designs section, ready to refine in BMR Studio

:::caution
A room scan is required before starting an AR Design. If no scan exists, the app will prompt you to scan the room first — this is essential for combining the AR-placed items with the room model.
:::
