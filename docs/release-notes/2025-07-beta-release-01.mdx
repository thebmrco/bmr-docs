# Release 1 (BETA) 2025 July

This is the first public beta release of the BETTERMEETINGROOMS solution.

---

## <Icon name="scan" size="1em" /> Room Capture & Measurement
- **Room scanning** using LiDAR technology
- **Marker placement** during scanning process
- **Acoustic measurement** using external speaker and baloon
---

## <Icon name="design" size="1em" /> Design & Collaboration
- **Design request** submission via the web platform
- **Advanced design view** on web including:
  - Audio coverage
  - Camera view
  - Screen coverage

---

## <Icon name="smartphone" size="1em" /> BMR Mobile & Platform
- BMR Mobile (iPhone/iPad) distributed via **TestFlight**
- Web platform for design, process tracking, and room insights

---

## <Icon name="wrench" size="1em" /> Configuration
- Full setup of:
  - Room
  - Location
  - Organization
- Configuration supported via both **BMR Mobile** and the **Web platform**

---

## <Icon name="users" size="1em" /> Onboarding
- Initial onboarding and organization setup

---

> ⚠️ **Note**
> All users must sign up via the **web** before logging into BMR Mobile.

---

Stay tuned for future updates and improvements!
