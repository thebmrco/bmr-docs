# Release 1.3 2026 June

This release makes it easier to **send feedback or ask for assistance** from both BMR Mobile and the Web, and brings two BMR Studio improvements: **mode-specific tabs** that cut visual clutter, and **Guidelines** that let you place recommended equipment directly into the design.

---

## Web Platform

<details className="bmr-details">
<summary className="bmr-summary">
  <span style={{display:"inline-flex", alignItems:"center", gap:"10px"}}>
    <Headset size={18} /> Report an Issue from the Web header
  </span>
</summary>

Reach the BMR team directly from any page on the Web — a **Report an Issue** button now sits in the header next to the room search. Pick a category (Bug / Feature Request / Question / Other), add a short summary and description, attach screenshots if useful, and optionally scope the report to an Organization, Location, or Room.

Reports go straight to the team's working queue for faster acknowledgement and resolution.

See **[Send Feedback](../getting-started/send-feedback)** for the full walkthrough, including the BMR Mobile entry point.

</details>

Each **BMR Studio** mode now shows only the tabs relevant to that mode, so you never page through controls that don't apply to what you're doing.

**What changed**

- **View** mode exposes inspection-related tabs only.
- **Structurals** mode focuses on wall, window, and door tools.
- **Equipment** mode keeps the catalog and placement tabs front and centre.
- **Cables** mode shows the cable-routing tools.

The result is a less crowded workspace and fewer accidental detours into tools that would not have worked from the current mode anyway. See the [BMR Studio guide](../lifecycle/design/bmr-studio) for the per-mode tool list.

</details>

<summary className="bmr-summary">

    <Lightbulb size={18} /> BMR Studio Guidelines — Place equipment directly
  </span>
</summary>

BMR Studio now surfaces **Guidelines** in Equipment mode — recommended placements drawn from the active **Default Room Configuration** and the room's measured geometry. From a Guideline you can **place equipment directly into the design with a single click**, instead of dragging items from the catalog and aligning them by hand.

**Why it matters**

- Faster designs — the most common items land where they should without manual positioning.
- Consistency with your organisation's standards — Guidelines respect the **Default Room Configuration** for the room's size band.
- You still keep full control — Guidelines are suggestions, not enforced placements. Override anything that does not fit the specific room.

See [Default Room Configuration](../lifecycle/design/default-room-configuration) for how the recommendations are defined.

</details>

Reach the BMR team directly from BMR Mobile — **Send Feedback** is in the **Settings** tab. Pick a category (Bug / Feature Request / Question / Other), add a short summary and description, and optionally scope the report to an Organization, Location, or Room. The last 24 hours of app logs are attached automatically.

Reports go straight to the team's working queue for faster acknowledgement and resolution.

</details>
