# Cisco Video Room Calculator Integration

BMR integrates with **Cisco's Video Room Calculator (VRC)**, enabling structured 2D room planning and Cisco device configuration directly within the BMR environment. Designs created through the integration can be previewed and explored in **Cisco Workspace Designer**.

---

## <Icon name="layout" size="1em" /> What It Does

The integration brings Cisco's official room planning capabilities into the BMR lifecycle workflow. Rather than switching between separate tools, you can plan Cisco device configurations from within BMR — with all versions, history, and room context kept in one place.

Through the integration you can:

- Edit rooms in **2D layout mode** using the Cisco Video Room Calculator
- **Add and configure Cisco collaboration devices** — endpoints, codecs, and peripherals
- Evaluate **recommended room system configurations** based on Cisco's official design logic
- Align device selection with **Cisco standards and performance guidelines**

---

## <Icon name="palette" size="1em" /> Preview in Cisco Workspace Designer

Designs created in BMR through the VRC integration can be **exported and previewed in Cisco Workspace Designer**. This gives you access to Cisco's extended visualization and configuration capabilities directly from your BMR design.

- View photorealistic room previews using Cisco's native rendering
- Use advanced Cisco-native planning and configuration tools
- Continue the workflow within the Cisco ecosystem without losing BMR as the system of record

BMR remains the central hub — version history, room data, and the Active Design are all tracked in BMR regardless of where the design is previewed.

---

## <Icon name="circle-check" size="1em" /> Lifecycle Integration

Unlike using the standalone Cisco tools, designs created via the VRC integration in BMR:

- Are saved as **structured versions** within the room
- Remain accessible directly from the **Room page**
- Can be compared against other design versions and historical configurations
- Stay connected to the **Active Design**, catalog components, and the approval workflow

This provides full design traceability and governance at portfolio level — across all your Cisco-equipped rooms.
