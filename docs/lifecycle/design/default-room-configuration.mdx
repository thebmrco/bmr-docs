# <Icon name="catalog" size="1em" /> Default Room Configuration

Default Room Configurations let organisation admins define **recommended equipment setups** based on room size and usage type. When a user designs a room in BMR Studio, matching configurations surface automatically — making it fast to apply consistent standards across an entire portfolio.

---

## <Icon name="circle-check" size="1em" /> How It Works

1. **Define a configuration** — set a name, room size range (minimum and maximum square meters), usage type, and a description.
2. **Select equipment** — browse the full BMR equipment catalog and pick the recommended items with quantities. Equipment is organised by category (displays, chairs, tables, video bars, etc.).
3. **Save** — the configuration is stored at the organisation level and applies to all rooms in the organisation.

When a user opens **BMR Studio** to design a room that falls within the defined size range, the recommended configuration appears in the sidebar under **Room guidelines**. The user can review the suggested items and place them directly into the design — or choose their own equipment instead.

---

## <Icon name="star" size="1em" /> Why It Matters

**Expert standards, at any scale.** Instead of relying on each designer to know the right setup for every room size, the organisation defines it once. Every room of a similar size gets the same recommendation — whether you're deploying 10 rooms or 10,000.

**Faster design.** Starting from a recommended configuration is significantly faster than browsing the full catalog from scratch. The user still has full flexibility to adjust, but the baseline is already there.

**Consistency.** Rooms of similar size and purpose get similar setups. This reduces rework, simplifies procurement, and makes support easier — because the equipment is predictable.

---

## <Icon name="scan" size="1em" /> Example

An organisation could define the following configurations:

| Configuration | Size Range | Suggested Equipment |
|---|---|---|
| Small Huddle | 10–20 sq m | Room Bar, single 55" display, 4 chairs, 1 table |
| Medium Meeting | 20–40 sq m | Room Bar, dual 65" displays, 6 chairs, 1 table |
| Large Boardroom | 40–80 sq m | Codec, 3 ceiling mics, dual 75" displays, 14 chairs, 1 table |

When a designer opens a 30 sq m room in BMR Studio, the "Medium Meeting" recommendation appears automatically.

---

## <Icon name="admin-settings" size="1em" /> Managing Configurations

Default Room Configurations are managed under the **Default Room Configurations** tab on the Organisation page at **[app.bettermeetingrooms.com](https://app.bettermeetingrooms.com)**.

Organisation admins can:

- Create new configurations
- Edit existing configurations (size range, equipment, quantities)
- Delete configurations that are no longer needed

Changes apply to all future design sessions across the organisation. Existing designs are not affected.
