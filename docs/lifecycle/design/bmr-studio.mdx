# <Icon name="palette" size="1em" /> BMR Studio

**BMR Studio** is the single 3D environment used for all room design work in BMR. It is accessible on the **Web**, **iPhone**, and **iPad**. BMR Studio is where you model rooms, place equipment, manage design versions, and mark a design as **Active**.

:::info Beta feature — available to all users
BMR Studio is currently in **Beta**, but enabled for everyone. A full public release is coming soon.
:::

## <Icon name="scan" size="1em" /> Modes

BMR Studio uses dedicated modes so you can focus on one thing at a time:

- **View** — explore the room and review designs. Navigate the 3D model, inspect placed equipment, and get an overview of the current design state.
- **Structurals** — adjust structural components such as walls, windows, doors, and other building elements. Modify the room geometry to match the real space.
- **Equipment** — add and remove equipment and furniture in the design. Browse the full catalog, place items, configure quantities, and position everything in the room.
- **Cables** — plan and document cable routing between devices. Visualise how cables run through the room and document infrastructure requirements.

More modes will be added over time, and existing modes will continue to be refined.

---

## <Icon name="equipment" size="1em" /> What You Can Do

- **Place and configure equipment** from the catalog — cameras, screens, speakers, codecs, furniture, and more
- **Adjust room structure** — walls, windows, doors, and other building elements
- **Plan cable routing** between devices
- **Create and compare multiple design versions** for the same room
- **Generate an equipment list** automatically from placed items
- **Move a design through approval** and mark it as Active when installed
- **Review Default Room Configuration recommendations** — when a room matches a defined size range, suggested equipment appears in the sidebar

## <Icon name="circle-check" size="1em" /> How to Open BMR Studio

### On the Web

1. Go to **[app.bettermeetingrooms.com](https://app.bettermeetingrooms.com)**
2. Open the room
3. Navigate to **Design**
4. Click **Open Studio** — the 3D editor loads with your room scan as the base

### On iPhone or iPad

1. Open the room in the BMR App
2. Go to the **Designs** tab
3. Open an existing design or create a new one — BMR Studio loads inside the app

## <Icon name="copy" size="1em" /> Design Versions

Each room can have **multiple design versions** in BMR Studio. This lets you:

- Explore different equipment configurations or layout options side by side
- Preserve historical versions as the room evolves
- Track which version was actually deployed

One design version can be designated as the **Active Design** — the authoritative record of the room as it currently exists. All other versions represent proposed or historical states.

---

## <Icon name="circle-check" size="1em" /> Reviewing and Approving a Design

Whether created in BMR Studio, via AR Design, or delivered by a BMR designer, the review process is the same:

1. Open the room's **Design** section to view available versions
2. Inspect the 3D layout, coverage visuals, and equipment list
3. Add comments or request adjustments
4. **Approve** the design when ready
5. Once physically installed, **mark it as Active** — it becomes the Active Design and the room's live record

## <Icon name="star" size="1em" /> Tips

:::tip
The richer your Discovery data — scan quality, marker coverage, acoustic measurements — the more accurately BMR Studio reflects the real room.
:::

- **Start with a scan** — BMR Studio uses the 3D scan as the foundation. A high-quality scan means more accurate placement.
- **Use modes** — switch between modes rather than trying to do everything at once. Each mode surfaces the tools relevant to that task.
- **Compare versions** — create multiple design versions to explore options before committing.
- **Check Default Room Configurations** — if your organisation has defined recommended setups, they appear automatically in the sidebar when the room size matches.
