# <Icon name="circle-check" size="1em" /> Deployed Design

:::info
The concept of the "Digital Twin" is now represented by the **Deployed Design** in BMR Studio. See the [Deploy Phase](../deploy/index.mdx) for full details.
:::

The **Deployed Design** is the design version in BMR Studio that has been approved and physically installed. It is the authoritative record of the room as it currently exists — the single source of truth for equipment, layout, cabling, and infrastructure.

Only one design version can be the Deployed Design at a time. All other versions represent proposed or historical states.

The Deployed Design is automatically tracked as designs move through the approval and deployment workflow in BMR Studio.
