## Delivered

<RoadmapItem icon={CircleCheck} title="BMR Studio">
  Web-based Studio environment for design management with dedicated modes (View, Structurals, Equipment, Cables). Available on Web, iPhone, and iPad.
</RoadmapItem>

<RoadmapItem icon={CircleCheck} title="Active Design & Lifecycle Flow">
  All designs versioned in BMR Studio with a clear Active Design as the authoritative room record.
</RoadmapItem>

<RoadmapItem icon={CircleCheck} title="AR Design">
  Real-time 3D placement with structured outputs linked to the Active Design. Create designs directly from AR.
</RoadmapItem>

<RoadmapItem icon={CircleCheck} title="Acoustic Workflow">
  Guided multi-position acoustic measurement with spatially anchored speaker locations, ambient noise chart, and Xi20/Xi30 metrics.
</RoadmapItem>

<RoadmapItem icon={CircleCheck} title="Compass">
  Room orientation captured during scan and visualised in BMR Studio.
</RoadmapItem>

<RoadmapItem icon={CircleCheck} title="Notes">
  Persistent notes on all elements — organisations, locations, rooms, scans, markers, acoustics, and designs. Available on Web and App.
</RoadmapItem>

<RoadmapItem icon={CircleCheck} title="Default Room Configuration">
  Organisation admins can define recommended equipment setups by room size and usage type. Recommendations surface in BMR Studio during design.
</RoadmapItem>

---

## Next (3–6 Months)

<RoadmapItem icon={Gauge} title="BMR Room Evaluation (Beta)">
  First version of a room evaluation engine that uses metrics to score how the technology is placed in a room and the experience it will deliver. Beta release with basic functionality.
</RoadmapItem>

<RoadmapItem icon={AudioLines} title="Advanced Acoustic Engine">
  Improved noise floor handling, smarter material recommendations, and higher measurement consistency.
</RoadmapItem>

<RoadmapItem icon={Palette} title="BMR Studio Mode Refinement">
  BMR Studio modes (View, Structurals, Equipment, Cables) are already available in the current version. Going forward, each mode will be refined significantly — clearer guidance, sharper tools, and tighter workflows per mode.
</RoadmapItem>

<RoadmapItem icon={Palette} title="BMR Studio Refinement">
  Improved Active Design validation, version handling, and design-to-deployment workflows.
</RoadmapItem>

<RoadmapItem icon={RefreshCw} title="Webex Room Synchronization — General Availability">
  Currently available as a Beta release for testing. The released version will bring stable two-way sync between BMR and Webex Control Hub for locations and rooms.
</RoadmapItem>

<RoadmapItem icon={AudioLines} title="Room Auralizations">
  Listen to simulated room acoustics to validate design decisions before deployment.
</RoadmapItem>

<RoadmapItem icon={ScanSearch} title="Improved Object Detection">
  Enhanced recognition accuracy and reduced manual corrections during capture.
</RoadmapItem>
