## Now – Release Launch (March – April)

<RoadmapItem icon={Copy} title="Deployed Design & Lifecycle Flow">
  All designs versioned in BMR Studio with a clear Deployed Design as the authoritative room record. Available in Beta — targeting public Release.
</RoadmapItem>

<RoadmapItem icon={Palette} title="BMR Studio">
  Web-based Studio environment for design management, version control, and deployment workflows. Available in Beta — targeting public Release.
</RoadmapItem>

<RoadmapItem icon={ScanSearch} title="AR Design Evolution">
  Real-time 3D placement with structured outputs linked to the Deployed Design. Available in Beta — targeting public Release.
</RoadmapItem>

<RoadmapItem icon={AudioLines} title="Acoustic Workflow Upgrade">
  Spatially anchored speaker and measurement locations with improved reliability. Available in Beta — targeting public Release.
</RoadmapItem>

<RoadmapItem icon={RefreshCw} title="Webex Room Synchronization">
  Automatic sync between BMR and Webex Control Hub for locations and rooms. Available in Beta — targeting public Release.
</RoadmapItem>

<RoadmapItem icon={Palette} title="Studio Enhancements">

</RoadmapItem>
