## <Icon name="search" size="1em" /> Discovery

Features for capturing the current state of a room.

| Feature | Platform | Internal | Beta | Release | Notes |
|---|---|---|---|---|---|
| [Scan](../lifecycle/discovery/scan-marker-guide) | iOS | ✅ β01 | ✅ β01 | 🗓 Planned | LiDAR-based 3D room scan. iPhone 12 Pro+ / iPad Pro 2020+. Screen dimming disabled since β08. |
| [Marker](../lifecycle/discovery/scan-marker-guide) | iOS | ✅ β01 | ✅ β01 | 🗓 Planned | Photo markers for outlets, displays, cables. Can be added independently of scan since β04. |
| Marker 360 | iOS | 🔧 In development | 🗓 Planned | 🗓 Planned | 360° photo capture for richer spatial context at marker locations. |
| Compass | iOS · Web | 🔧 In development | 🗓 Planned | 🗓 Planned | Captures room orientation during scan. Displayed in BMR Studio to inform lighting and layout decisions. |
| Improved object detection | iOS | 🔧 In development | 🗓 Planned | 🗓 Planned | Enhanced recognition accuracy and reduced manual corrections during room capture. |
| Room measurements | iOS · Web | ✅ β03 | ✅ β03 | 🗓 Planned | Floor area, volume, and dimensions calculated from scan data and shown in the room overview. |
| [Acoustic measurement](../acoustics/acoustics-guide) | iOS | ✅ β01 | ✅ β01 | 🗓 Planned | Sweep (recommended) and balloon/impulse methods. Returns RT60 + MOS score. Guided flow added β08. Spatial position recording added β09. |

## <Icon name="design" size="1em" /> Design

Features for planning what a room should become.

| Feature | Platform | Internal | Beta | Release | Notes |
|---|---|---|---|---|---|
| BMR Studio | Web | ✅ β09 | ✅ β09 | 🗓 Planned | Single web-based 3D environment for room design. Multiple guided modes expose only relevant tools per workflow. |
| Design comparison *(legacy)* | Web | ✅ β04 | ✅ β04 | 🗓 Planned | Side-by-side comparison of designs in the previous design view. Covers acoustics, layout, and equipment differences. Not available in BMR Studio. |
| Design comparison in BMR Studio | Web | 🗓 Planned | 🗓 Planned | 🗓 Planned | Side-by-side view mode comparison within BMR Studio. Planned for a future release. |
| AR design | iOS | ✅ β01 | ✅ β01 | 🗓 Planned | Real-time 3D equipment placement via LiDAR on iOS. Design can be continued in BMR Studio on web. |
| Cisco VRC | Web | ✅ Available | 🔀 Toggle | 🗓 Planned | Video Room Calculator — Cisco's 2D room design tool, integrated as a second design path focused on Cisco devices. |
| Active design | Web | ✅ β09 | ✅ β09 | 🗓 Planned | The approved, installed design — the authoritative record of the room as it currently exists. Used as the reference for maintenance, troubleshooting, and future design cycles. |
| Room auralizations | Web | 🗓 Planned | 🗓 Planned | 🗓 Planned | Simulated room acoustics to validate design decisions before deployment. On the 3–6 month roadmap. |
